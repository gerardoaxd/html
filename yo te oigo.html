Puntos Clave para Entender un Sitio HTML en un Bloc de Notas

Cuando abres un archivo HTML en un bloc de notas, estás viendo el "esqueleto" de una página web. Aquí te decimos qué buscar para entender su estructura y contenido.

---

1. La Estructura Fundamental: El Esqueleto

Siempre busca estas etiquetas iniciales, definen el tipo y el alcance del documento:

  • <!DOCTYPE html>: Declara que es un documento HTML5. Siempre al principio.
  • <html>: La etiqueta raíz que contiene todo el contenido de la página.
  • <head>: Contiene información meta sobre la página (no visible directamente).
  • <body>: Aquí está todo el contenido visible de la página (texto, imágenes, etc.).
<!DOCTYPE html>
<html>
  <head>
    <title>Título de la Pestaña</title>
  </head>
  <body>
    <!-- Contenido visible aquí -->
  </body>
</html>
---

2. Contenido de Texto: Títulos y Párrafos

El texto se organiza con estas etiquetas:

  • <h1> a <h6>: Encabezados de diferentes niveles de importancia (h1 es el más importante).
  • <p>: Para párrafos de texto normal.
<h1>Mi Título Principal</h1>
<p>Este es un párrafo de texto en mi página.</p>
---

3. Interacción: Enlaces e Imágenes

Así se conectan las páginas y se muestran los elementos visuales:

  • <a href="url_destino">Texto del enlace</a>: Un enlace (hipervínculo). href indica a dónde va.
  • <img src="ruta_imagen.jpg" alt="Descripción">: Inserta una imagen. src es la fuente y alt es el texto alternativo.
<a href="https://www.ejemplo.com">Visita Ejemplo.com</a>
<img src="mi_imagen.jpg" alt="Una bonita imagen">
---

4. Organización: Listas y Tablas

Para estructurar datos de forma ordenada:

  • Listas no ordenadas (viñetas): <ul> para la lista, <li> para cada elemento.
  • Listas ordenadas (numeradas): <ol> para la lista, <li> para cada elemento.
  • Tablas: <table> (tabla), <tr> (fila), <th> (encabezado de celda), <td> (celda de datos).
<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>

<table>
  <tr>
    <th>Encabezado</th>
  </tr>
  <tr>
    <td>Dato</td>
  </tr>
</table>
---

5. Estilos y Funcionalidad: CSS y JavaScript

Aunque no veas el código completo, verás los "lazos" a estos archivos:

  • CSS (Estilos): Se enlaza en el <head> con <link rel="stylesheet" href="estilos.css">. Le da la apariencia.
  • JavaScript (Interactividad): Se enlaza con <script src="script.js"></script>. Añade dinamismo y funcionalidades.
<!-- En el head -->
<link rel="stylesheet" href="styles.css">

<!-- Antes de cerrar el body -->
<script src="main.js"></script>
---

6. Comentarios: Notas para el Desarrollador

Esto no es visible en la página, solo para los que leen el código:

  • <!-- Este es un comentario -->: Texto ignorado por el navegador.
<!-- Esto es un comentario útil para entender el código -->

Al entender estos puntos clave, puedes empezar a "leer" cualquier sitio web HTML directamente desde un simple bloc de notas, ¡y desentrañar cómo está construido!
